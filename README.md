# Bright-Car-Sales-Analysis
This project analyzes car sales data from Bright Motors to uncover insights that can improve sales performance, optimize inventory, and support business growth. The analysis focuses on understanding revenue drivers, customer preferences, and regional performance using data analytics and visualization tools.

Main Objective
The objective of this project is to identify top-performing car models, analyze pricing and sales patterns, evaluate regional performance, and provide data-driven recommendations to increase profitability and efficiency.
